# Frontend Mentor - Newsletter sign-up form with success message solution

This is a solution to the [Newsletter sign-up form with success message challenge on Frontend Mentor](https://www.frontendmentor.io/challenges/newsletter-signup-form-with-success-message-3FC1AZbNrv). Frontend Mentor challenges help you improve your coding skills by building realistic projects.

### The challenge

Users should be able to:

- Add their email and submit the form
- See a success message with their email after successfully submitting the form
- See form validation messages if:
  - The field is left empty
  - The email address is not formatted correctly
- View the optimal layout for the interface depending on their device's screen size
- See hover and focus states for all interactive elements on the page

### What I learned

I am happy I completed the project without struggling too much and my js skill were up to the challenge. 

I learned how to use the dialog HTML element with for the success message and open and close it with js.

I also learned how to get the user input from the form in js and use regex to validate the email. The limitations of using regex quickly became apparent and I would like to try use a third party package next time.
